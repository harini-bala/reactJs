# Data Flow Diagram (DFD) for Todo App

## Level 0 (Context Diagram)
```
+------------------+
|      User       |
+------------------+
         ↕
+------------------+
|    Todo App     |
+------------------+
         ↕
+------------------+
|  Task Management |
+------------------+
```

## Level 1 (Detailed Flow)
```
+------------------+
|      User       |
+------------------+
         ↓
+------------------+
|   Input Field   |
+------------------+
         ↓
+-------------------------+
|    State Management    |
|------------------------|
| - task (current input) |
| - tasks (task array)   |
+-------------------------+
         ↓
+-------------------------+
|    Task Operations     |
|------------------------|
| - Add Task            |
| - Delete Task         |
+-------------------------+
         ↓
+-------------------------+
|    Task Display        |
|------------------------|
| - Task List           |
| - Delete Buttons      |
+-------------------------+
```
